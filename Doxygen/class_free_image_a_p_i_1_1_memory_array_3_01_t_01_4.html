Detailed Description

Represents unmanaged memory, containing an array of a given structure.

Template Parameters
TStructuretype represented by the instance.

System.Boolean and System.Char can not be marshalled.

Use System.Int32 instead of System.Boolean and System.Byte instead of System.Char.
